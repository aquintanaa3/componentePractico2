# componentePractico2
h1 Nombre del Proyecto: NutriCampus - Sistema de Reserva y Gestión de Almuerzos para el Comedor Universitario.

Integrantes: Carlos Alvarado, Joel Jiménez, Adrian Quintana, Tony Macias

Descripción general del sistema propuesto: Se trata de una plataforma web y móvil diseñada para optimizar el servicio del comedor universitario. A través de la aplicación los estudiantes y el personal de la institución pueden visualizar el menú diario, reservar su almuerzo con anticipación, realizar pagos digitales y validar su ticket mediante un código QR en el punto de entrega.
